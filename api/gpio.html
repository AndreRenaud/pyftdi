Classes¶

class pyftdi.gpio.GpioController¶

GPIO controller for an FTDI port

close()¶

Close the FTDI interface.

direction¶

Reports the GPIO direction.

Returns:a bitfield specifying the FTDI GPIO direction, where high level reports an output pin, and low level reports an input pin
Return type:int
is_connected¶

Reports whether a connection exists with the FTDI interface.

open_from_url(url, direction, **kwargs)¶

Open a new interface to the specified FTDI device in bitbang mode.

Parameters:
  • url (str) – a FTDI URL selector
  • direction (int) – a bitfield specifying the FTDI GPIO direction, where high level defines an output, and low level defines an input
read_port()¶

Read the GPIO input pin electrical level.

Parameters:value (int) – a bitfield of GPIO pins. Each bit represent a GPIO pin, matching the logical input level of the pin.
set_direction(direction)¶

Update the GPIO pin direction.

Parameters:direction (int) – a bitfield of GPIO pins. Each bit represent a GPIO pin, where a high level sets the pin as output and a low level sets the pin as input/high-Z.
write_port(value)¶

Set the GPIO output pin electrical level.

Parameters:value (int) – a bitfield of GPIO pins.
