gpio - GPIO API¶

Direct drive GPIO pins of FTDI device.

Note

This mode is mutually exclusive with advanced serial MPSSE features, such as I2C, SPI, JTAG, …

If you need to use GPIO pins and MPSSE interface on the same port, you need to use the dedicated API. For now, this shared mode is only supported with the SPI API.

Quickstart¶

See tests/gpio.py example
